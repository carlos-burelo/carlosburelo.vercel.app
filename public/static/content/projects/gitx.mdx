## Introducción

GitX es una herramienta de terminal de linea de comandos con la finalidad de optimizar la velocidad en la cual trabajamos con [git](https://git-scm.com/) a traves de una serie de comandos simples y muy cortos los cuales ya cuentan con las instrucciones comunmente utilizadas como los commits entre otros.

## Motivación

En el tiempo que he estado adiministrando la creación de proyectos grandes y pequeños me he visto en el problema de tener que escribir los tipicos `git add .` y `git commit -m "some description"`y en algun momento pense, "que pasaria si reduzco al minimo lo que debo escribit para hacer un commit" y ahi nacio la idea de crear GitX, una herramienta de terminal donde utilizaria los identificadores mas cortos posibles para entender la logica del comando y con la capacidad de reducir la tarea repetitiva que es registrar cambios.

# Instalación

La instalación de esta herramienta es muy sencilla ya que la podemos instalar a traves de [`npm`](https://www.npmjs.com/package/gitx-app) o [`yarn`](https://yarnpkg.com/package/gitx-app) como una dependencia cualquiera.

```bash
npm install -g gitx-app
# ó
yarn global add gitx-app
```

# Uso

La utilización de esta herramienta es muy sencillo, solo hay que tomar como referencia que se utilizan las primeras letras de cada palabra que conforma el comando (_hay excepciones_) y este por detras realiza la tarea asignada, por ejemplo.

```bash
gi # --> git init --> inicializa un nuevo repositorio git
ga # --> giti add . --> Agrega todos los cambios existentes.
gc # --> git commit -m "some" --> registra el comit con una descripción
...
```

# Comandos basicos

| Atajo | Comando        | Descripción                              | Ejemplo                   |
| ----- | -------------- | ---------------------------------------- | ------------------------- |
| ga    | `git add .`    | Agrega todos los cambios                 | `ga index.ts`             |
| gc    | `git commit`   | Registra el cambio con un commit         | `gc add index.ts `        |
| gcl   | `git clone`    | Clona el repositorio                     | `gcl carlos-burelo/gitx ` |
| gi    | `git init`     | Inicializa in repositorio de git         | `gi`                      |
| gs    | `git status`   | Muestra el estatus de git                | `gs`                      |
| gp    | `git push`     | Empuja los cambios al repositorio remoto | `gp main`                 |
| gr    | `git remote`   | Agrega un repositorio remoto             | `gr http:...`             |
| gb    | `git branch`   | Muestra las ramas locales                | `gb prod`                 |
| gch   | `git checkout` | Cambia de rama                           | `gch main`                |
